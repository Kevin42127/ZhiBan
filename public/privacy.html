隱私政策

智伴 ZhiBan Chrome 擴充功能

最後更新日期：2025年12月16日

1. 資料收集聲明

智伴（ZhiBan）重視您的隱私權。本擴充功能採用最小化資料收集原則，僅收集提供服務所必需的資料。

重要：本擴充功能不會收集以下類型的資料：
  • 個人識別資訊（姓名、地址、電子郵件、年齡、身分證字號等）
  • 健康資訊
  • 財務和付款資訊
  • 驗證資訊（密碼、憑證等）
  • 個人通訊資料（電子郵件、簡訊等）
  • 位置資訊
  • 網頁記錄
  • 使用者活動（點擊、滑鼠位置、鍵盤操作等）
  • 網站內容

2. 我們使用的資料

2.1 本地存儲資料

本擴充功能使用 Chrome 瀏覽器的本地存儲功能（chrome.storage.local）來保存您的對話歷史記錄。這些資料：

  • 僅存儲在您的本地瀏覽器中
  • 不會上傳到任何外部伺服器
  • 不會與第三方分享
  • 您可以隨時透過擴充功能介面清空這些資料

2.2 傳輸的資料

為了提供 AI 對話服務，您的訊息內容會傳輸到我們的後端 API 伺服器（https://zhiban.vercel.app）進行處理。這些資料：

  • 僅用於即時處理以提供 AI 回應
  • 不會被永久保存或存儲
  • 不會被用於分析、廣告或其他目的
  • 所有傳輸都透過 HTTPS 加密連線

3. 資料使用目的

我們使用資料的唯一目的是提供智能對話服務：

  • 處理您的訊息並提供 AI 回應
  • 保存對話歷史以便您隨時回顧（僅在本地）
  • 提供上下文理解功能

我們不會將您的資料用於：

  • 廣告或行銷目的
  • 分析用戶行為
  • 出售或轉移給第三方
  • 與產品單一目的無關的任何用途
  • 確定信用評級或貸款相關用途

4. 資料安全

我們採取多項安全措施保護您的資料：

  • HTTPS 加密：所有通訊都透過安全的 HTTPS 連線傳輸
  • API 金鑰驗證：後端使用 API 金鑰驗證機制，防止未授權存取
  • 速率限制：實施多層速率限制，防止濫用和保護服務穩定
  • 本地存儲：對話記錄僅存儲在您的本地瀏覽器中，不會上傳到伺服器

5. 第三方服務

5.1 Google Material Icons

本擴充功能使用 Google Material Icons 字體庫來顯示圖標。此服務由 Google 提供，我們不會將您的資料傳輸給 Google。

5.2 Groq API

本擴充功能的後端使用 Groq API 來提供 AI 對話服務。您的訊息內容會傳輸到 Groq API 進行處理，但我們不會將您的個人識別資訊傳輸給 Groq。

6. 您的權利

您對自己的資料擁有完全的控制權：

  • 查看資料：您可以隨時在擴充功能中查看您的對話歷史
  • 刪除資料：您可以隨時透過擴充功能介面清空所有對話記錄
  • 停止使用：您可以隨時卸載擴充功能，所有本地資料將被清除

7. 兒童隱私

本擴充功能不針對 13 歲以下的兒童。我們不會故意收集兒童的個人資訊。如果您是家長或監護人，發現您的孩子向我們提供了個人資訊，請聯繫我們，我們將立即刪除這些資訊。

8. 隱私政策變更

我們可能會不定期更新本隱私政策。任何變更都會在本頁面上發布，並更新「最後更新日期」。我們建議您定期查看本頁面以了解最新的隱私政策。
