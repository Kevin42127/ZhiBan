# 智伴 ZhiBan - Chrome 擴充功能說明

## 商品描述

智伴（ZhiBan）是一款智能陪伴聊天助理 Chrome 擴充功能，整合先進的 AI 技術，為你提供即時、自然、友善的對話體驗。無論是工作上的問題、學習上的疑惑，還是日常生活的陪伴，智伴都能成為你最可靠的智能助手。

## 核心功能

### 🤖 智能對話助理
- **即時回應**：快速、流暢的對話體驗，就像與朋友聊天一樣自然
- **自然語言處理**：理解你的問題，提供準確、有用的回答
- **上下文理解**：記住對話歷史，提供連貫的回應
- **智能語言判斷**：自動識別中文和英文，用對應語言回應

### 💬 流暢對話體驗
- **多行輸入**：支援長文本輸入，輕鬆表達複雜問題
- **即時流式回應**：AI 回應即時顯示，無需等待
- **對話歷史保存**：自動保存所有對話記錄，隨時回顧
- **簡潔介面設計**：專注對話，無干擾的使用體驗

### 🎨 優雅的使用者介面
- **現代化設計**：簡潔美觀的對話介面
- **響應式布局**：適配不同螢幕尺寸
- **流暢動畫**：自然的互動體驗
- **直觀操作**：一鍵清空對話，輕鬆管理

### 🔒 安全可靠
- **API 金鑰保護**：後端安全處理，保護你的資料
- **HTTPS 加密**：所有通訊透過安全連線
- **隱私保護**：對話記錄僅存儲在本地

## 開始使用

1. **安裝擴充功能**：點擊「加到 Chrome」完成安裝
2. **點擊圖標**：在瀏覽器工具列點擊智伴圖標
3. **開始對話**：輸入你的問題，立即獲得 AI 回應
4. **享受便利**：隨時提問，隨時獲得協助
