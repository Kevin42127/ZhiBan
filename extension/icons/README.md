# 圖標檔案

請在此資料夾中放置以下圖標檔案：

- `icon16.png` - 16x16 像素，單色設計
- `icon48.png` - 48x48 像素，單色設計  
- `icon128.png` - 128x128 像素，單色設計

## 圖標設計建議

- 使用單色設計（符合使用者要求）
- 建議使用 AI 或聊天相關的圖示
- 確保在不同尺寸下都清晰可見
- 可以使用線上工具如 [Favicon Generator](https://favicon.io/) 或 [RealFaviconGenerator](https://realfavicongenerator.net/) 生成

## 快速生成方式

可以使用以下方式快速生成圖標：

1. 使用線上圖標生成器
2. 使用設計軟體（如 Figma、Photoshop）設計後匯出
3. 使用命令行工具如 ImageMagick 調整尺寸
